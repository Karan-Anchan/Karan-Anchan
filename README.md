### 🚀 &nbsp; Featured Projects

#### 🧬 &nbsp; [UNETR — 3D Abdomen Segmentation](https://github.com/KaranAnchan/Unetr_3D_Abdomen_Segmentation)
*PyTorch · MONAI · Vision Transformers*

3D segmentation pipeline using **UNETR** (Vision Transformer encoder + CNN decoder) on the BTCV dataset. HU-windowing and isometric resampling on DICOM / NIFTI files. Class-imbalance handled via Dice-Cross Entropy loss.

> **Result:** Mean Dice **0.82** across **13 organs**.

#### 🗣 &nbsp; [English → Hindi Neural Machine Translation](https://github.com/KaranAnchan/en-hi-nmt-transformer)
*PyTorch · Transformers · Gradio*

A **43M-parameter Transformer encoder-decoder built from scratch in PyTorch** (no `nn.Transformer`), trained on **500k AI4Bharat Samanantar** parallel sentence pairs.

- **Tokenizer:** byte-level BPE
- **Optimizer:** Adam + Noam learning-rate schedule, label smoothing
- **Decoding:** length-normalized beam search (GNMT formula)
- **Evaluation:** frozen 5,000-pair held-out test set
- **Deployment:** Gradio demo with live cross-attention heatmaps

> **Result:** **BLEU 16.93** / **chrF++ 41.58** / **TER 71.41**
